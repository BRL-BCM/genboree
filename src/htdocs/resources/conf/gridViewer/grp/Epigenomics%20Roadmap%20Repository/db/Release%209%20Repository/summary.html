• Data Access Policy
  • Data embargo period: from 04/15/2013 - 01/15/2014 or earlier as specified here
  • Select cells by clicking and dragging, then use "View Selections" in the Selections menu to view selections in the Atlas Gene Browser or the UCSC Genome Browser
  • Use "Save Selections" in the Selections menu to save selected (highlighted) cells in a group and database of your choice (requires login)
  • To see data authors, other metadata, and to download data, click a sample name in the first column or an assay type in the header row
  • Expression Array data may be downloaded here
  • Human Epigenome Atlas releases are intended to be cumulative: e.g. Release 3 includes all Release 2 data and additional submissions
